== Description ==

Product filter for woocommerce product plugin sets woocommerce product sku search, woocommerce product name search, woocommerce product category filter, woocommerce product attribute filter at the product listing woocommerce shop page. 

Woocommerce product sku search, woocommerce product name search, woocommerce product category filter, woocommerce product attribute filter is fully manageable by store admin using product filter for woocommerce product plugin. 

Woocommerce product sku search, woocommerce product id search, woocommerce product category filter, woocommerce product attribute filter can be placed using widgets at woocommerce product listing page, shop page.

- Admin setings: Backend > Products > Search & filter settings.

- For frontend: Backend > Appearance > Widgets > WPS Product Filter > Drag & drop at sidebar.

Lets take a look which we offered for you.

= Key Features =

-- 1. Search By SKU and Name etc. Its a one page AJAX search.
Store Admin has the capability to play with the following settings.
- Enable / Disable.
- Title, before the search box.
- Product listing view (Grid / List / Grid - List Switcher ).
- Placeholder of search box.
- Add To Cart / Read More button settings.
- Loader Image Changes ( 10 sample images ).

-- 2. Filter by Product Attribute available to your store.
- Enable / Disable any of available attribute.
- Type of filter ( ColorBox / CheckBox / RadioBox / DropDown) for each attribute.
- Reset Enable / Disable for each filter.

-- 3. Filter By Category available to your store.
- Enable / Disable.
- Title of filter.
- Type of category listing style. ( Hierchical / List / DropDown).
- Hide Empty Category ( Yes / No).
- Show Product Count ( Yes / No).
- Order By Category ( ID - Ascending or Desecnding / Name - Ascending or Desecnding).

-- 4. Add Custom CSS.

For more details please visit [official plugin page at WPSuperiors.](https://www.wpsuperiors.com/woocommerce-product-filter/)

== Installation ==

= Minimum Requirements =
* WooCommerce 2.5 requires WordPress 4.1+
* WooCommerce 2.6 requires WordPress 4.4+

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Product Filter For WooCommerce Product, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Product Filter For WooCommerce Product” and click Search Plugins. Once you’ve found our plugin.You can install it by simply clicking “Install Now”.
And activate it.

*Keep In Mind, Before activating “Product Filter For WooCommerce Product” activate “WooCommerce”
*Premium and Basic Version not active at the same time.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Goto Product->WPSPF Filter Settings.
4. Widgets -> Drag WPSPF Products Filter wdiget and Drop it into Shop Page's Sidebar
